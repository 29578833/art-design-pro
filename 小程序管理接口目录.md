## 1. 轮播图管理（5 个）

| 接口地址                                        | 请求方式 | 接口名称       |
| ----------------------------------------------- | -------- | -------------- |
| `/adminapi/scrap/config/banner_list.html`       | GET      | 轮播图列表     |
| `/adminapi/scrap/config/banner_save.html`       | POST     | 新增轮播图     |
| `/adminapi/scrap/config/banner_update/:id.html` | POST     | 修改轮播图     |
| `/adminapi/scrap/config/banner_delete/:id.html` | DELETE   | 删除轮播图     |
| `/adminapi/scrap/config/banner_sort/:id.html`   | POST     | 调整轮播图排序 |

## 3. 协议管理（2 个）

> 走系统设置协议接口，不是 scrap 的 config/agreement\_\*：

| 接口地址 | 请求方式 | 接口名称 | 说明 |
| --- | --- | --- | --- |
| `/adminapi/setting/get_agreement/:type.html` | GET | 获取协议内容 | type：4 用户服务协议 / 3 隐私政策 / 5 报废回收服务协议 |
| `/adminapi/setting/save_agreement.html` | POST | 设置协议内容 | 保存协议 |
